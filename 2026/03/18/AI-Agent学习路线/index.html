学习路径概览

I. Python 全栈工程 (基石篇)

1. Python 语法精髓 ✅

  • ✅ 类型推导与基础语法: 现代 Python 核心语法。
  • ✅ 切片与字符串操作: 高效文本预处理技术。
  • ✅ 推导式与生成器: 内存优化与流式数据处理。
  • ✅ Match-Case 结构化匹配: 复杂 Prompt 协议解析。
  • ✅ 装饰器基础: 日志监控、权限控制与 AOP 实战。
  • ✅ 魔法方法与特殊协议: 深入理解 Python 对象协议。
  • ✅ Dataclass 数据类: 结构化数据建模。

2. Python 进阶工程化 ✅

  • ✅ 面向对象 OOP: 面向对象基础、继承与多态、元类Metaclass、组合与 AI 框架设计模式。
  • ✅ 并发编程: GIL与多线程、Asyncio异步、多进程并行。
  • ✅ 元编程: 元类Metaclass、描述符协议、抽象基类ABC。

II. 数据科学与大模型内核 (理论篇) 🔄

1. 数据科学三剑客 ✅

  • ✅ NumPy: NumPy张量计算。
  • ✅ Pandas: Pandas数据处理。
  • ✅ Matplotlib: Matplotlib数据可视化。

2. 深度学习基础 🔄

  • ✅ PyTorch 核心: PyTorch深度学习基础。
  • ✅ Transformer 架构: Transformer架构解析。
  • ✅ 推理优化: 推理优化与量化（KV Cache、FlashAttention、PagedAttention、量化技术）。

3. Prompt Engineering

  • CoT 思维链: Chain-of-Thought 推理增强。
  • Few-Shot: 少样本学习与 In-Context Learning。
  • 结构化输出: JSON/XML 协议约束。

III. RAG 检索增强架构 (实战篇) 🔄

1. RAG 基础与向量检索

  • ✅ RAG 基础: RAG检索增强与向量检索（RAG架构、Embedding、向量数据库、完整Pipeline）。

2. 文档智能解析

  • ✅ 文档解析: 文档智能解析（PDF解析、OCR、表格提取、LangChain Loader）。

3. 检索增强技术

  • ✅ 混合检索与重排序: 混合检索与重排序（BM25+Vector、Rerank、Query改写、RAGAS评测）。

  • ✅ GraphRAG: GraphRAG知识图谱检索（知识图谱构建、Neo4j、图谱+向量融合）。

IV. Agent 智能体编排 (巅峰篇)

1. Agent 核心机制

  • Planning: 思维链 (CoT)、ReAct 模式、自我反思。
  • Memory: 短期上下文管理、长期向量记忆。
  • Tool Use: Function Calling 原理与工具库扩展。

2. 编排框架与协同

  • LangChain: LangChain基础（核心组件、Chain编排、PromptTemplate、Memory）。
    • Prompt模板与输出解析：输入模板（PromptTemplate/ChatPromptTemplate/FewShot）+ 输出解析（StrOutputParser/JsonOutputParser/PydanticOutputParser）。
    • LCEL链式编排：`|` 管道语法、Runnable 接口、四种调用方式、RunnablePassthrough/Lambda/Parallel、高级编排模式。
    • Memory记忆管理：ChatMessageHistory、RunnableWithMessageHistory、多会话隔离、Token 爆炸处理、持久化存储。
    • Tool与Agent：Tool 定义（@tool/StructuredTool）、Function Calling 机制、Agent 创建、ReAct 模式。
  • LangGraph: LangGraph状态机编排：State/Node/Edge、条件边、循环、Supervisor 多 Agent 协作、Human-in-the-Loop、持久化。
  • LangChain4J: 企业级 Java Agent 开发。
  • MCP 协议: Model Context Protocol 跨模型上下文协议。

3. 多智能体系统

  • 多代理协作: Supervisor 模式、层级编排。
  • SOP 流程控制: 企业级标准作业程序自动化。

V. 多模态交互 (实战篇)

1. 语音交互

  • ASR: 流式语音识别、实时转写。
  • TTS: 流式语音合成、情感表达。
  • 打断机制: 全双工对话、智能插话。

2. 多模态理解与生成

  • Vision-Language: 图文理解、视觉问答。
  • 多模态生成: 图像生成、视频生成。

VI. 工程化部署 (生产篇)

1. 环境管理

  • Poetry: 依赖管理、虚拟环境。
  • Conda: 环境隔离、包管理。
  • Docker: 容器化部署、镜像构建。

2. 模型部署

  • Transformers: HuggingFace 库、模型加载。
  • vLLM: 高性能推理、PagedAttention。

3. 训练与微调

  • 小模型训练: BERT 预训练、下游任务微调。
  • LoRA: 大模型微调、PEFT 技术。

4. 可观测性

  • RAGAS: RAG 评测体系、 Faithfulness 指标。
  • OpenInference: 可观测性、链路追踪。
  • LakeFS: 数据版本管理。
