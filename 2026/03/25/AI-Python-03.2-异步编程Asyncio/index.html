AI-Python-03.2: 异步编程 - Asyncio

总结

asyncio 核心概念

概念 说明
async def 定义协程函数
await 等待协程完成
asyncio.run() 运行主协程
asyncio.gather() 并发执行多个协程
asyncio.create_task() 创建后台任务
asyncio.wait() 灵活等待任务

异步同步原语

原语 等价同步版本
asyncio.Lock threading.Lock
asyncio.Semaphore threading.Semaphore
asyncio.Queue queue.Queue
asyncio.Event threading.Event
asyncio.Condition threading.Condition

asyncio vs threading

特性 asyncio threading
并发模型 协程（单线程） 多线程
切换开销 低 中等
适用场景 I/O 密集型 I/O 密集型
调试难度 中等 高
库支持 需要 async 版本 所有库

AI 开发最佳实践

  1. I/O 密集型用 asyncio: HTTP 请求、数据库查询
  2. 使用信号量控制并发: 避免过载
  3. 流式处理大数据: async for + AsyncIterator
  4. 阻塞操作用 to_thread: 避免阻塞事件循环
  5. 合理使用连接池: 复用资源
