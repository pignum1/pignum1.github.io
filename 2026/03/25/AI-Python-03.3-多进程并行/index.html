总结

进程 vs 线程 vs 协程

特性 进程 线程 协程
内存 独立 共享 共享
GIL 不受影响 受影响 受影响
并行 真正并行 伪并行 伪并行
开销 高 中 低
适用 CPU 密集 I/O 密集 I/O 密集

进程池最佳实践

进程间通信方式

方式 适用场景
Queue 生产者-消费者
Pipe 双向通信
Value/Array 简单共享数据
Manager 复杂共享对象

AI 开发最佳实践

  1. CPU 密集型用多进程: 数据预处理、特征提取
  2. 使用进程池: 避免频繁创建销毁进程
  3. 减少进程间通信: 通信有开销
  4. 注意内存使用: 每个进程独立内存
  5. initializer 初始化: 每个进程加载一次模型
