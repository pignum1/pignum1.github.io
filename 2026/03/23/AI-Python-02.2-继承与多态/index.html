MRO 查找规则 (C3 算法)：

  1. 子类优先于父类
  2. 多个父类按定义顺序查找
  3. 保证一致性（不会出现A 在 B 之前又在 B 之后的情况）

继承与多态对照表

特性 Python Java
单继承 class B(A) class B extends A
多继承 class C(A, B) 不支持，使用接口
调用父类 super() super
抽象类 ABC 模块 abstract class
抽象方法 @abstractmethod abstract
方法重写 自动支持 @Override
多态 鸭子类型 接口/继承

AI 开发中的典型应用

  • 模型继承: 基础模型 → 特定任务模型
  • 数据加载器: DataLoader 基类 → CSV/JSON/Parquet 加载器
  • 混入类: LoggerMixin、SerializableMixin
  • 策略模式: 不同算法通过多态切换

设计原则

  1. 优先使用组合 而非继承
  2. 多用接口 (Python 中是抽象基类)
  3. 少用多继承，避免复杂性
  4. 遵循 LSP (里氏替换原则)
