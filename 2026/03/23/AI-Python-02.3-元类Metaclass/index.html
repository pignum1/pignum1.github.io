总结

元类对照表

特性 Python Java
动态创建类 type() 反射 + 字节码库
元类 metaclass 无直接等价物
抽象类 ABC 模块 abstract class
抽象方法 @abstractmethod abstract
描述符 __get__/__set__ 无直接等价物

AI 开发中的典型应用

  • ORM 框架: Django ORM、SQLAlchemy
  • 模型注册: PyTorch 模型自动注册
  • 配置验证: 配置类属性验证
  • 单例模式: 数据库连接池

设计原则

  1. 能用简单方式解决就不用元类
  2. 元类会增加代码复杂度
  3. 优先考虑装饰器和描述符
  4. 确保代码可读性

1.3 type vs Java Object

可以把 type 理解成 Java 里的 Object 超类：

Python Java 说明
type Object 所有类的根
class X: class X extends Object 都隐式关联
type(X) X.class 获取类信息
