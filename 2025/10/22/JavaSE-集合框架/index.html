JavaSE - 集合框架 (Collections Framework)

Java 集合框架（JCF）是一个用来存储和操作对象集合的统一架构。它包含了一系列的接口、实现类和算法，极大地简化了对数据集合的处理。这是 JavaSE 中最核心、最重要的部分之一。

本篇笔记将作为集合框架学习的索引和枢纽，您可以点击以下链接，跳转到各个核心接口的详细学习笔记中。

2. 各核心接口详解 (点击跳转)

  • List 接口: 有序、可重复的集合。重点：ArrayList vs. LinkedList

    • ArrayList: 基于动态数组，查询快，增删慢。
    • LinkedList: 基于双向链表，增删快，查询慢。

  • Set 接口: 无序、不可重复的集合。重点：HashSet, TreeSet

    • HashSet: 基于 HashMap，无序，性能最高。
    • LinkedHashSet: 维持插入顺序。
    • TreeSet: 基于红黑树，自动排序。

  • Map 接口: 键值对集合。重点：HashMap, ConcurrentHashMap

    • HashMap: 基于哈希表，无序，性能高。
    • LinkedHashMap: 维持插入或访问顺序。
    • TreeMap: 基于红黑树，键自动排序。
    • ConcurrentHashMap: 线程安全且高效的 Map。

  • Queue 接口: 先进先出(FIFO)的队列。重点：ArrayDeque, PriorityQueue

    • ArrayDeque: 高效的队列和栈实现。
    • PriorityQueue: 基于堆的优先队列。

  • 树形结构: 深入理解 TreeSet 和 TreeMap 的底层原理：红黑树

    • 二叉树、平衡二叉搜索树 (AVL, 红黑树) 的概念。
