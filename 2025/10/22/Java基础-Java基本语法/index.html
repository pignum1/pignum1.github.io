Java 基础语法

Java 语言作为一门面向对象的编程语言，其基础语法是构建一切复杂应用的基础。本章将系统回顾 Java 的核心语法要素。

1. 关键字与标识符

  • 关键字 (Keywords): Java 语言预先定义并赋予特殊含义的单词，如 public, class, static, void, int, if, else, for, while 等。关键字不能作为标识符使用。
  • 标识符 (Identifiers): 程序员在程序中自定义的名称，用于命名变量、类、方法、包等。命名规则：
    • 由字母、下划线 _ 或美元符号 $ 开头。
    • 后续字符可以是字母、数字、下划线 _ 或美元符号 $。
    • 不能是 Java 关键字。
    • 区分大小写。
    • 建议遵循“驼峰命名法”：类名首字母大写，变量名和方法名首字母小写，常量全大写并用下划线分隔。

2. 变量与数据类型

2.1 变量 (Variables)

变量是程序中存储数据的基本单元。在使用前必须先声明后赋值。

  • 声明: 数据类型 变量名; (例如: int age;)
  • 赋值: 变量名 = 值; (例如: age = 25;)
  • 声明并赋值: 数据类型 变量名 = 值; (例如: String name = "Alice";)

2.2 数据类型 (Data Types)

Java 的数据类型分为两大类：基本数据类型 (Primitive Data Types) 和引用数据类型 (Reference Data Types)。

基本数据类型 (8 种)
类型 占用字节 取值范围 默认值 示例
byte 1 -128 到 127 0 byte b = 10;
short 2 -32768 到 32767 0 short s = 1000;

  • 注意: long 类型赋值时建议加 L 或 l 后缀，float 类型赋值时必须加 F 或 f 后缀。
引用数据类型

除了 8 种基本数据类型，其他所有类型都是引用数据类型，包括：

  • 类 (Class): String, Scanner, Object 等。
  • 接口 (Interface)
  • 数组 (Array)

3. 运算符 (Operators)

Java 支持多种运算符，用于执行各种操作。

  • 算术运算符: +, -, *, /, % (取模), ++ (自增), -- (自减)
  • 关系运算符: ==, !=, >, <, >=, <= (结果为 boolean)
  • 逻辑运算符: && (与), || (或), ! (非)
    • && 和 || 具有短路特性：如果第一个操作数已经能确定结果，则不再计算第二个操作数。
  • 位运算符: & (按位与), | (按位或), ^ (按位异或), ~ (按位非), << (左移), >> (右移), >>> (无符号右移)
  • 赋值运算符: =, +=, -=, *=, /=, %= 等
  • 三元运算符: 条件 ? 表达式1 : 表达式2 (如果条件为 true，则结果为 表达式1；否则为 表达式2)

4. 流程控制 (Control Flow)

流程控制语句用于控制程序执行的顺序。

4.1 选择结构

  • if-else 语句:

  • switch 语句:

    • switch 表达式可以是 byte, short, char, int (JDK 1.7+ 支持 String, enum)
    • break 语句用于跳出 switch 块，否则会“穿透”到下一个 case。

4.2 循环结构

  • for 循环:

  • while 循环:

  • do-while 循环:

  • 增强 for 循环 (foreach): 用于遍历数组或集合。

4.3 跳转语句

  • break: 终止当前循环或 switch 语句。
  • continue: 跳过当前循环的剩余部分，进入下一次循环。
  • return: 结束方法执行，并返回一个值（如果方法有返回值）。

5. 数组 (Arrays)

数组是存储同类型数据集合的容器。

  • 声明: 数据类型[] 数组名; 或 数据类型 数组名[];
  • 创建: 数组名 = new 数据类型[长度]; (例如: int[] numbers = new int[5];)
  • 初始化: 数据类型[] 数组名 = {值1, 值2, ...}; (例如: String[] names = {"Alice", "Bob"};)
  • 访问: 数组名[索引] (索引从 0 开始)

6. 方法 (Methods)

方法是执行特定任务的代码块。它封装了可重用的逻辑。

  • 定义: [修饰符] 返回值类型 方法名([参数列表]) { // 方法体 }
  • 调用: 对象名.方法名([参数列表]); 或 类名.方法名([参数列表]); (对于静态方法)

7. 注释 (Comments)

注释用于解释代码，提高可读性，不会被编译器执行。

  • 单行注释: // 这是单行注释
  • 多行注释: /* 这是多行注释 */
  • 文档注释: /** 这是文档注释 */ (可用于生成 API 文档)
