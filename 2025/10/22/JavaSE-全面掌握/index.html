JavaSE 全面掌握

Java SE (Standard Edition) 是 Java 语言的核心和基础。精通 Java SE 是学习 Java EE、Android 开发以及各种高级框架（如 Spring、MyBatis）的前提。本部分旨在全面、系统地巩固 Java 的核心知识体系，为后续深入学习 JVM、并发编程和框架源码打下坚实的基础。

学习路径

我们将遵循一条从基础到进阶的路径，逐步深入 Java 的世界：

  1. 基础语法: 回顾变量、数据类型、运算符、流程控制等基础知识
  2. 面向对象编程 (OOP): 深入理解封装、继承、多态三大特性，以及类、对象、接口、抽象类的设计与实现
  3. 常用 API: 掌握 `String`、`StringBuilder`、`Math`、日期时间 API (`LocalDate`, `LocalDateTime`) 等核心类的使用
  4. 集合框架 (Collections Framework): 这是 JavaSE 的重中之重。我们将深入学习 `List`, `Set`, `Map` 等核心接口及其常用实现类（如 `ArrayList`, `LinkedList`, `HashSet`, `HashMap`）的原理、源码和适用场景。
  5. 异常处理 (Exception Handling): 学习 Java 的异常体系，掌握 `try-catch-finally`、`throws` 等关键字的使用，以及如何设计健壮的异常处理代码。
  6. I/O 与 NIO: 学习传统的输入/输出流（InputStream/OutputStream, Reader/Writer），并深入了解 NIO (New I/O) 带来的非阻塞 I/O 模型。
  7. 泛型 (Generics): 理解泛型的概念、优势，以及类型擦除、通配符等关键机制。
  8. 反射 (Reflection): 学习在运行时动态地获取类信息和操作对象的能力，这是许多框架实现其功能的底层技术。
  9. 注解 (Annotations): 了解 Java 内置注解，并学习如何自定义注解。
  10. 多线程基础: 初步接触 Thread 类和 Runnable 接口，为后续深入学习并发编程做准备。
  11. Java 8+ 新特性: 重点学习 Lambda 表达式、Stream API、Optional 类等，这些新特性可以极大地提升编码效率和代码可读性。

让我们从最基础的部分开始，一步一个脚印，全面掌握 JavaSE。
