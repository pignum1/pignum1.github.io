详细学习计划

I. 核心 Java 与 JVM 精通 (深度与原理)

1. 计算机基础

  • 计算机操作系统: 进程线程模型、内存管理、文件系统。
  • 计算机组成原理:
    • 系统概论、数据表示与运算。
    • 指令系统、CPU结构、控制器设计。
    • 存储层次、Cache 原理。
    • 虚拟存储、IO 控制 (DMA/中断)。
    • 多处理器与缓存一致性 (MESI)。
  • 计算机网络:
    • OSI 七层与 TCP/IP 四层。
    • TCP 三次握手、四次挥手、滑动窗口、拥塞控制。
    • HTTP/HTTPS、HTTP/2、TLS 握手原理。

2. Java 语言基础与高级特性

  • 核心语法、OOP 精髓、集合框架源码级分析。
  • I/O 模型: BIO、NIO (Selector/Buffer/Channel)、AIO 的原理与 Netty 预研。
  • 反射、自定义注解、泛型擦除与通配符。
  • Java 8+ 特性: Lambda、Stream API、Optional、CompletableFuture。

3. JVM 内部原理

  • JMM 内存布局: 堆、栈、元空间、直接内存。
  • GC 深度解析: 三色标记法、G1/ZGC 收集器原理、STW 治理。
  • 类加载: 双亲委派模型、破坏双亲委派 (SPI/OSGi)、自定义类加载器。
  • 调优实战: Arthas、JProfiler、MAT 内存泄漏排查、GC 日志分析。

4. 并发编程与多线程 (JUC)

  • 可见性、原子性、有序性、`volatile` 原理。
  • 锁升级: synchronized 偏向/轻量/重量锁、ReentrantLock (AQS 源码)。
  • JUC 工具包: 线程池深度调优、CountDownLatch、CyclicBarrier、Semaphore、ConcurrentHashMap。
  • 高频面试突击。

II. 高级 Java 生态系统与框架

1. Spring 全家桶源码级开发

  • Spring 核心: IoC 循环依赖解决方案、AOP 动态代理 (JDK/CGLIB)。
  • Bean 生命周期源码。
  • Spring MVC 执行链路。
  • Spring Boot Starter 机制、SPI 机制。
  • Spring Cloud Alibaba:
    • Nacos: 注册中心 Distro 协议、配置中心长轮询。
    • Sentinel: 滑动窗口限流、熔断降级、系统自适应保护。
    • Seata: 分布式事务 AT、TCC、Saga 模式源码分析。

2. 消息队列 (MQ)

  • Kafka: 零拷贝、顺序写、ISR 机制、控制器选举、Rebalance 治理。
  • RabbitMQ: AMQP 协议、死信队列、镜像队列。
  • RocketMQ: 事务消息、定时消息、CommitLog 存储模型。

III. 数据管理 (Data Management)

  1. 关系型数据库 (MySQL)

    • InnoDB 存储引擎: B+ 树索引、MVCC、Undo/Redo Log。
    • 事务与锁: 隔离级别、行锁、间隙锁 (Gap Lock)、死锁检测。
    • SQL 优化: 执行计划 (Explain)、慢查询治理、深度分页优化。
    • 架构: 读写分离、ShardingSphere 分库分表实战。

  2. NoSQL 数据库

    • Redis: 跳表物理结构、持久化 (RDB/AOF)、Cluster 哈希槽机制。

IV. 系统设计与架构 (System Design)

  1. 分布式系统理论

    • 核心协议: CAP/BASE、Raft 共识算法、分布式锁 (Redis/ZK)、Snowflake 及其大厂改良 (Leaf)。

  2. 高并发架构实战

    • 挡、疏、导技术: LVS/Nginx 负载均衡、令牌桶/漏桶限流、**Alibaba Canal Binlog 同步**、多级缓存。

  3. 案例设计与分析

    • 千万级秒杀: Lua 脚本扣减、MQ 削峰、热点分片、请求合并。
    • 百万级长连接: Netty 长连接、扩散风暴防御、聚合推送策略。
    • 异地多活: 单元化设计 (Unitization)、流量闭环、数据双向同步冲突解决。

V. 云原生与 DevOps

  • 云原生架构: Docker 镜像瘦身、K8s 调度、Ingress 入口治理、Istio 服务网格、可观测性 (Prometheus/SkyWalking)。

VI. 大数据技术 (Big Data)

  • 实时计算: Flink 状态编程、窗口机制、Exactly-once 保证。
  • 存储: Apache Iceberg 湖仓一体原理。

VIII. 算法与数据结构 (Algorithms & Data Structures)

  1. 核心算法: 动态规划 (DP)、红黑树、Trie 树、布隆过滤器、滑动窗口、广度优先搜索 (BFS)。
  2. 冲刺: 针对 LeetCode 大厂高频题进行分类突击。

IX. 软技能与工程实践

  1. 技术领导力: 方案设计能力、Code Review 体系、新人导师制度。
  2. 故障排查: 复杂线上故障复盘、全链路压测、AI 稳定性 Runbook 编写。
