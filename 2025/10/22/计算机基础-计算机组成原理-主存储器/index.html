主存储器 (Main Memory)

主存储器（简称主存或内存）是计算机存储器层次结构中的核心组成部分，位于高速缓存（Cache）和辅助存储器（硬盘）之间。它用于存放 CPU 当前正在执行的程序和处理的数据。CPU 可以直接对主存进行读写访问。

现代计算机的主存主要由半导体存储器构成，根据其工作原理的不同，主要分为两大类：静态随机存取存储器 (SRAM) 和动态随机存取存储器 (DRAM)。

1. 静态随机存取存储器 (SRAM - Static RAM)

  • 工作原理: SRAM 使用触发器 (Flip-flop) 来存储每一位（bit）的数据。一个触发器通常由 6 个晶体管构成。只要有持续的电源供应，触发器就能一直保持其状态（0 或 1），无需刷新。

  • 特点:

    • 速度快: 由于其结构简单，无需刷新操作，SRAM 的存取速度非常快，接近 CPU 的速度。
    • 价格昂贵: 每个 bit 需要多个晶体管，集成度低，单位容量的成本非常高。
    • 功耗较大: 在保持数据时，触发器一直处于工作状态，因此功耗相对较高。

  • 主要用途: 因其高速特性，SRAM 主要用于制造高速缓存 (Cache) 和 CPU 内部的寄存器 (Registers)。

2. 动态随机存取存储器 (DRAM - Dynamic RAM)

  • 工作原理: DRAM 使用电容 (Capacitor) 来存储每一位的数据。电容中存储电荷代表 1，不存储电荷代表 0。由于电容会随时间自然放电（漏电），其中的数据会逐渐消失，因此 DRAM 需要一个定期的刷新 (Refresh) 操作来重新充电，以保持数据不丢失。

  • 特点:

    • 速度较慢: 相对于 SRAM，DRAM 的读写速度较慢，并且需要额外的刷新周期。
    • 价格便宜: 每个 bit 只需要一个晶体管和一个小电容，集成度非常高，单位容量的成本低廉。
    • 功耗较小: 相比 SRAM，功耗更低。

  • 主要用途: 因其高集成度和低成本的优势，DRAM 主要用于制造计算机的主存储器（内存条）。

3. SRAM 与 DRAM 的总结对比

特性 SRAM (静态RAM) DRAM (动态RAM)
存储元件 触发器 (6T) 电容 (1T1C)
刷新操作 不需要 需要（周期性刷新）
存取速度 快 (ns 级别) 慢 (几十 ns 级别)
集成度 低 高
单位成本 高 低
功耗 大 小
主要应用 高速缓存 (Cache), 寄存器 主内存 (内存条)

4. 只读存储器 (ROM - Read-Only Memory)

除了 RAM，存储器中还有一类是只读存储器 (ROM)。它的特点是数据一旦写入，就不能轻易更改，并且断电后数据不会丢失（非易失性）。

  • 主要用途: ROM 通常用于存放计算机启动时所需的基本程序和配置信息，例如 BIOS (Basic Input/Output System) 或 UEFI 固件。

  • 类型演变:

    • MROM (Mask ROM): 掩模式 ROM，在制造时就将数据写入，无法更改。
    • PROM (Programmable ROM): 可编程 ROM，允许用户用专门的设备写入一次。
    • EPROM (Erasable PROM): 可擦除可编程 ROM，可以用紫外线照射来擦除数据，然后重新编程。
    • EEPROM (Electrically EPROM): 电可擦除可编程 ROM，可以用电信号来擦除和重写数据。我们现在常用的 闪存 (Flash Memory) 就是一种特殊的 EEPROM，它被广泛用于 U 盘和固态硬盘 (SSD)。

5. 主存与 CPU 的连接

主存储器通过总线（Bus）与 CPU 相连，主要包括：

  • 地址总线 (Address Bus): 用于指定 CPU 要访问的内存单元的地址。地址总线的位数决定了 CPU 可寻址的最大内存空间（例如，32 位地址总线可以寻址 2^32 = 4GB 的空间）。
  • 数据总线 (Data Bus): 用于在 CPU 和主存之间双向传输数据。数据总线的位数（总线宽度）决定了单次能传输的数据量，是影响性能的关键因素。
  • 控制总线 (Control Bus): 用于传输控制信号，如读/写信号、时钟信号等，以协调 CPU 和主存之间的操作。
