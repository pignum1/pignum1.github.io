Java基本语法

Java作为一门强类型、面向对象的语言，其基本语法是构建所有程序的基础。本章将详细介绍Java中的变量、数据类型、运算符、流程控制语句和数组。

1. 变量与数据类型

变量是程序中存储数据的基本单元。Java是强类型语言，这意味着每个变量都必须声明其数据类型。

1.1 变量

  • 定义： 变量是内存中的一块区域，用于存储数据。
  • 声明与初始化：
  • 命名规则：
    • 由字母、数字、下划线 _ 和美元符号 $ 组成。
    • 不能以数字开头。
    • 不能是Java关键字。
    • 区分大小写。
    • 建议使用驼峰命名法（camelCase），例如 firstName。

1.2 数据类型

Java的数据类型分为两大类：基本数据类型（Primitive Data Types）和引用数据类型（Reference Data Types）。

1.2.1 基本数据类型 (8种)
类型 占用字节 默认值 取值范围
byte 1 0 -128 到 127
short 2 0 -32768 到 32767
int 4 0 -2^31 到 2^31-1 (约 ±20亿)
long 8 0L -2^63 到 2^63-1
float 4 0.0f 单精度浮点数
double 8 0.0d 双精度浮点数 (默认浮点类型)
char 2 '\u0000' 单个Unicode字符 (0 到 65535)
boolean 1 (JVM) false true 或 false

注意：

  • long 类型的值后面需要加 L 或 l (建议大写L)。
  • float 类型的值后面需要加 F 或 f (建议小写f)。
1.2.2 引用数据类型

引用数据类型包括类（Class）、接口（Interface）、数组（Array）等。它们存储的是对象的引用（内存地址），而不是实际的数据值。

2. 运算符

运算符用于对变量和值进行操作。

2.1 算术运算符

运算符 描述 示例
+ 加法 a + b
- 减法 a - b
* 乘法 a * b
/ 除法 a / b
% 取模 a % b
++ 自增 a++ (后增), ++a (前增)
-- 自减 a-- (后减), --a (前减)

2.2 关系运算符 (比较运算符)

运算符 描述 示例
== 等于 a == b
!= 不等于 a != b
> 大于 a > b
< 小于 a < b
>= 大于等于 a >= b
<= 小于等于 a <= b

2.3 逻辑运算符

运算符 描述 示例
&& 逻辑与 (短路与) A && B
|| 逻辑或 (短路或) A || B
! 逻辑非 !A

2.4 位运算符

用于对二进制位进行操作。

运算符 描述
& 按位与
| 按位或
^ 按位异或
~ 按位非
<< 左移
>> 右移
>>> 无符号右移

2.5 赋值运算符

运算符 描述 示例
= 简单赋值 a = 10
+= 加并赋值 a += b 等同于 a = a + b
-= 减并赋值 a -= b 等同于 a = a - b
*= 乘并赋值 a *= b 等同于 a = a * b
/= 除并赋值 a /= b 等同于 a = a / b
%= 模并赋值 a %= b 等同于 a = a % b

2.6 三元运算符 (条件运算符)

条件 ? 表达式1 : 表达式2
如果条件为真，则返回表达式1的值；否则返回表达式2的值。

3. 流程控制语句

流程控制语句用于控制程序执行的顺序。

3.1 选择结构 (条件语句)

3.1.1 if 语句
3.1.2 switch 语句

适用于基于一个变量的不同值执行不同代码块的场景。

注意： break 语句用于跳出 switch 块。如果没有 break，程序会继续执行下一个 case 的代码（称为“穿透”）。

3.2 循环结构

3.2.1 for 循环
3.2.2 while 循环

先判断条件，条件为真则执行循环体。

3.2.3 do-while 循环

先执行一次循环体，再判断条件。至少执行一次。

3.3 跳转语句

  • break: 终止当前循环或 switch 语句。
  • continue: 跳过当前循环的剩余部分，进入下一次循环。
