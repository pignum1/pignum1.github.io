Java开发环境搭建

搭建Java开发环境是学习Java的第一步，它包括安装JDK（Java Development Kit）和选择一个集成开发环境（IDE）。

1. JDK的安装与配置

JDK是Java开发的核心，包含了Java运行时环境（JRE）和开发工具。

1.1 下载JDK

访问Oracle官方网站或OpenJDK社区网站下载适合您操作系统的最新稳定版JDK。

  • Oracle JDK: https://www.oracle.com/java/technologies/downloads/
  • OpenJDK: https://openjdk.java.net/ (推荐，通常更开放和免费)

选择与您操作系统（Windows, macOS, Linux）和架构（x64, ARM64等）匹配的版本。

1.2 安装JDK

  • Windows: 双击下载的.exe安装包，按照提示一步步完成安装。建议安装到默认路径，或者一个不包含空格和特殊字符的路径（例如 C:\Java\jdk-xx）。
  • macOS: 双击下载的.dmg文件，按照提示完成安装。
  • Linux: 通常可以通过包管理器（如apt for Debian/Ubuntu, yum for CentOS/RHEL）安装OpenJDK，或者下载.tar.gz包手动解压。

1.3 配置环境变量

配置环境变量是为了让操作系统能够找到Java的命令（如java和javac）。

  • Windows:
    1. 右键“此电脑” -> “属性” -> “高级系统设置” -> “环境变量”。
    2. 在“系统变量”下，点击“新建”，变量名为 JAVA_HOME，变量值为JDK的安装路径（例如 C:\Program Files\Java\jdk-xx）。
    3. 找到“Path”变量，双击编辑，点击“新建”，添加 %JAVA_HOME%\bin。
    4. 点击“确定”保存所有更改。
  • macOS/Linux:
    1. 打开终端。

2. 常用IDE的安装与使用

集成开发环境（IDE）提供了代码编辑、编译、调试等一站式服务，极大地提高了开发效率。

2.1 IntelliJ IDEA (推荐)

IntelliJ IDEA是JetBrains公司开发的一款功能强大、智能的Java IDE，分为社区版（免费）和旗舰版（付费）。

  • 下载： 访问JetBrains官网 https://www.jetbrains.com/idea/download/ 下载社区版。
  • 安装： 按照安装向导一步步完成安装。
  • 基本使用：
    1. 启动IDEA，选择“New Project”。
    2. 选择“Java”，确保JDK已正确配置。
    3. 选择“Command Line App”或“Empty Project”开始。
    4. 在项目中创建新的Java类文件（.java）。
    5. 编写代码，点击运行按钮即可编译和执行。

2.2 Eclipse

Eclipse是一款开源、免费的Java IDE，拥有庞大的插件生态系统。

  • 下载： 访问Eclipse官网 https://www.eclipse.org/downloads/ 下载Eclipse IDE for Java Developers版本。
  • 安装： 解压下载的压缩包即可使用，无需传统安装。
  • 基本使用：
    1. 启动Eclipse，选择一个工作空间（Workspace）。
    2. 选择“File” -> “New” -> “Java Project”。
    3. 在项目中创建新的Java类文件。
    4. 编写代码，点击运行按钮即可。
