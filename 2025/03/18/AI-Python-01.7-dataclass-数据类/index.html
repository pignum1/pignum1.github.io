// Java 泛型对比
import java.util.*;

public class Node {
T value;
Node left;
Node right;

public Node(T value) {
    this.value = value;
    this.left = null;
    this.right = null;
}

public void setLeft(Node<T> left) {
    this.left = left;
    if (left != null) {
        left.right = this.right;
    }
}

public void setRight(Node<T> right) {
    this.right = right;
    if (this.right != null) {
        this.right = left;
    }
}

public boolean hasLeft() {
    return this.left != null;
}

public boolean hasRight() {
    return this.right != null;
}

public <T> getValue() {
    return this.value;
}

public static <T> int traverse(Node<T> node) {
    if (node == null) {
        return 0;
    }
    int count = 1 + traverse(node.left) + traverse(node.right);
    return count;
}

## 总结

本文详细介绍了 Python 的 dataclass 数据类：

1. **基础语法**
   - `@dataclass` 装饰器
   - 自动生成 `__init__` 初始化方法
   - 自动生成 `__repr__` 字符串表示
   - 自动生成 `__eq__` 等价比较
   - 自动生成 `__hash__` 哈希值

2. **参数默认值**
   - `field: type = default` 语法
   - 支持 `frozen=True` 不可变类
   - 支持 `*args` 可变参数

3. **智能特性**
   - 自动类型提示支持
   - `typing` 模块集成
   - `__post_init__` 初始化后处理

4. **实战应用**
   - 模型配置：定义 LLM 配置
   - 数据验证：类型注解
   - 配置管理：集中式配置

与 Java 相比，Python 的 dataclass 提供了更简洁的语法和更强的表达能力，特别适合 AI 开发中的数据模型定义：

| 特性 | Python 3.7+ | Java 16+ |
|------|----------|-------|
| 基础语法 | 自动生成方法 | 需手动实现 |
| 参数默认值 | field: value | 需手动写 setter |
| 可变参数 | *args 支持 | 不支持 |
| 类型提示 | typing 模块 | 无原生 |
| 不可变类 | frozend=True | final 类不可变 |
| 集成支持 | dataclass | 无内置 |
| 后初始化 | __post_init__ | 构造函数 |
| 内存效率 | __slots__ | 无 |
