总结

本文详细介绍了 Python 的装饰器：

  1. 基础语法

    • @decorator 语法
    • 函数嵌套：@decorator 相当于 decorator(func)
    • @functools.wraps(func) 可以简化装饰器定义

  2. 带参数的装饰器

    • 可以接受参数定制装饰器行为
    • repeat(n) 装饰器重复执行函数 n 次
    • preserve_metadata 装饰器保留函数元数据

  3. 多个装饰器堆叠

    • 支持从上到下依次应用装饰器
    • @decorator1 + @decorator2 = decorator2(decorator1(function))

  4. 类装饰器

    • 使用 __init__ 和 cls.__init__ 装饰整个类
    • @class_decorator 装饰器类
    • __init__ 中使用 cls 指代 self

  5. 高级特性

    • functools.wraps 简化装饰器定义
    • @wraps() 一次性添加多个装饰器
    • @functools.cached_property 缓存属性
    • @functools.lru_cache(maxsize) 缓存结果
    • @functools.partial 固定参数

  6. 实战应用

    • 日志装饰器：自动记录函数调用信息
    • 缓存装饰器：减少重复计算
    • 限流装饰器：限制调用频率
    • 监控装饰器：添加性能监控

与 Java 相比，Python 的装饰器提供了更简洁的语法和更强的表达能力，特别适合：

  • 装饰器语法：@decorator 比 Java 的代理类更简洁
  • 参数处理：Python 自动解包参数更简单
  • 类装饰器：__init__ 装饰器比 Java 类继承更直观
  • 函数式：functools 提供了函数式装饰器支持
