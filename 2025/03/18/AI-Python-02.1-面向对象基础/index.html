总结

本文详细介绍了 Python 的面向对象基础：

  1. 类定义

    • class 关键字
    • __init__ 构造函数
    • 实例方法

  2. 类与实例

    • 类：数据结构定义
    • 实例：类的具体对象
    • self 引用实例

  3. 封装与访问控制

    • 公有属性：直接访问 self.field
    • 私有方法：_field 开头（约定）
    • @property 装饰器：控制访问

  4. 方法类型

    • 实例方法：self 参数，访问实例
    • 类方法：@classmethod，cls 参数，访问类
    • 静态方法：@staticmethod，无特殊参数
