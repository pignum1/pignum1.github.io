总结

本文详细介绍了 Python 的推导式与生成器：

  1. 列表推导式 - 简洁的列表创建语法，支持条件和嵌套

    • 基础语法: [expression for item in iterable if condition]
    • 带条件过滤
    • 嵌套推导式
    • 性能优于传统 for 循环

  2. 字典推导式 - 一行代码创建字典

    • 基础语法: {key_expr: value_expr for item in iterable}
    • 支持条件过滤
    • 支持复杂的键值表达式

  3. 集合推导式 - 自动去重

    • 基础语法: {expr for item in iterable}
    • 自动去重
    • 支持条件过滤

  4. 生成器表达式 - 使用圆括号的惰性求值

    • 语法: (expr for item in iterable)
    • 节省内存
    • 只在需要时计算

  5. 生成器函数 - 使用 yield 关键字的惰性生成

    • yield 返回值并暂停执行
    • 节省内存
    • 支持提前终止

  6. 生成器高级特性

    • send() - 向生成器发送值
    • throw() - 向生成器抛出异常
    • close() - 关闭生成器并触发清理
    • yield from - 委托给子生成器

  7. 实战应用

    • 日志文件分析
    • 数据清洗管道
    • AI 数据预处理
    • 批量数据库查询

与 Java 相比，Python 的推导式和生成器提供了更简洁的语法和更高效的内存管理，特别适合处理大数据流场景。
