总结

本文详细介绍了 Python 的 match-case 结构化匹配：

1. 基础语法

  • match value: case pattern: 结构
  • _ 通配符作为默认情况
  • 支持复杂的模式嵌套

2. 类型模式

  • 内置类型：int(), str(), float(), list(), dict()
  • 组合类型：int() | float(), str() | list()
  • None 检查：case None

3. 序列模式

  • 列表模式：[], [first], [first, *rest]
  • 嵌套列表：[[a, b], [c, d]]
  • 元组模式：(x, y), (x, y, z)
  • 字符串模式："hello", ["h", "e", "l", "o"]

4. 映射模式

  • 字典键模式：{"key": value}, {"k1": v1, "k2": v2}
  • 嵌套字典：{"data": {"key": "details": details} as info}

5. 守卫条件

  • case pattern if condition: 语法
  • 支持多个条件：if age >= 18 and age < 65

6. OR 模式

  • 使用 | 组合多个模式："hello" | "hi" | "hey"

7. as 模式绑定

  • 给子模式命名：case (x, y) as coords
  • 嵌套绑定：{"user": {"details": details} as user_details}

8. 实战应用

  • LLM 工具调用解析
  • RAG 检索结果处理
  • Agent 状态机转换
  • HTTP 响应解析

与 Java 对比

特性 Python 3.10+ Java 21+
原生支持时间 2019年 2023年
模式表达力 强大 逐步增强
类型解构 自动 record 模式
嵌套匹配 殷洁 较复杂
纯程 惰性高 阻塞
as 绑定 原生支持 手动实现

match-case 特别适合 AI 开发中的数据解析、Prompt 处理、状态机转换等场景。
